# Project Rules

Use the /browse skill from gstack for all web browsing instead of default browser tools.
Available gstack commands:
- /office-hours         - Challenge feature logic & map edge cases
- /plan-ceo-review     - Evaluate ideas using YC evaluation methodologies
- /plan-eng-review     - Draft detailed architectural specifications
- /review               - Review branch code changes before committing
- /qa                   - Boot up Playwright to navigate & test staging URLs
- /ship                 - Merge, sync with main, and automate PR creation
